Swagdoku is sudoku version implemented entirely in python. To open the file, you will first require some dependencies:
	Pygame
	Copy
These sudokus are rather easy, and in future versions, (if i ever get around to it) there will be difficulty function which varies based on strategies needed to solve. To run, simply launch the sudokuboard.py in a terminal with the following command:

python3 (pathname)/swagdoku/sudokuboard.py

On non mac machines, use the equivalent python command. Updates will be posted to https://github.com/lsgordon. Make sure to download the whole folder for intended results.

You can start a new board with the "newgame" button. You can move with arrow keys.
